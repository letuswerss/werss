# 设置页

## 1. 页面作用

设置页面向普通用户，提供应用偏好设置和部分系统配置的可视化入口。用户无需修改配置文件，即可在 Web UI 中调整行为配置。

## 2. 入口路径

- 路径：`/settings`
- 导航栏点击"设置"进入

## 3. 页面结构

页面按功能分为多个卡片分组：

### 3.1 应用设置卡片

| 配置项 | 说明 |
|---|---|
| 显示水印 | 开关，在页面上显示版权水印信息 |
| 暗色模式 | 开关，切换暗色/亮色主题 |
| 采集数据起始时间 | 日期选择器，设置公众号文章采集的起始日期，系统从该日期开始抓取 |

### 3.2 文章定期清理卡片

控制是否自动删除旧文章（需配合 `server.enable_job=true` 和 `-job True` 启动参数）：

| 配置项 | 说明 |
|---|---|
| 启用定期清理 | 总开关 |
| 保留最近 | 数字输入，保留多少天内的文章 |
| 时间基准 | 下拉选择：`created_at`（入库时间）或 `publish_time`（微信发布时间） |

**注意**：
- 默认保留最近 7 天
- 删除方式与 `article.true_delete` 配置一致
- 若设置 `WERSS_ENV_OVERRIDES_DB=true`，则未在环境变量 / yaml 中留空的项才会使用此处写入数据库的值

### 3.3 图片与对象存储卡片

控制是否将公众号文章图片转存到 MinIO：

| 配置项 | 说明 |
|---|---|
| 转存公众号图片到 MinIO | 总开关 |

三种状态：
- **MinIO 未启用或不可用**：不转存，使用微信侧图片链接
- **MinIO 可用、开关关闭**：不转存，节省存储但外链可用性受上游访问策略影响
- **MinIO 可用、开关开启**：下载并上传到对象存储，使用存储 URL（更稳定）

### 3.4 标签提取设置卡片

**位置**：设置页最下方区域（定时抓取任务卡片上方）

管理 AI 自动提取标签的提示词模板：

| 配置项 | 说明 |
|---|---|
| 标签提取提示词 | Textarea，支持占位符 `{{max_tags}}`、`{{custom_tags_hint}}`、`{{text}}` |
| 预设模板 | 下拉选择：科技、医学、法律、自定义 |

**预设说明**：
- **科技**：默认提示词，优先提取公司名、产品名、技术名词
- **医学**：优先提取疾病名、药物名、治疗方案、科室
- **法律**：优先提取法条名、案由、程序节点、机构名
- **自定义**：用户自行编辑的提示词

保存后直接影响后续标签提取结果。

### 3.5 定时抓取任务卡片

显示调度器状态和消息任务列表：

| 区域 | 说明 |
|---|---|
| 调度器状态 | 显示"运行中"（绿色）或"未运行"（灰色），包含任务数量 |
| 定时任务列表 | 显示每个消息任务的名称、状态（已启用/已禁用）、Cron 表达式、下次执行时间 |
| 刷新状态按钮 | 重新加载调度器信息 |

**前提条件**：需服务以 `-job True` 启动且 `server.enable_job=true`，调度器才会运行。

## 4. 常见操作

### 4.1 修改采集起始时间

1. 点击日期选择器，选择日期
2. 点击"保存"按钮
3. 系统将从该日期开始采集文章（不影响已采集的历史文章）

### 4.2 配置文章清理策略

1. 开启"启用定期清理"开关
2. 设置保留天数（如 7 天）
3. 选择时间基准
4. 点击"保存"

### 4.3 修改标签提取提示词

1. 从预设模板中选择（如"医学"）
2. 或选择"自定义"后直接编辑 Textarea
3. 点击"保存提示词"按钮

### 4.4 刷新调度器状态

点击"刷新状态"按钮，重新获取调度器运行状态和消息任务下次执行时间。

## 5. 注意事项

- 设置页中保存的配置项（除水印、暗色模式外）均写入数据库 `config_management` 表
- 若 `WERSS_ENV_OVERRIDES_DB=true`，则以环境变量和 `config.yaml` 为准，数据库值仅在对应键未设置或为空时生效
- 标签提取提示词保存后只影响新提交的文章，不追溯已提取的标签
- 文章定期清理需要同时满足：开关开启 + `-job True` 启动 + `server.enable_job=true`

## 6. 常见报错 / 空态

| 场景 | 表现 |
|---|---|
| 保存失败 | Toast 显示错误信息 |
| 标签提取提示词为空 | 阻止保存，提示"标签提取提示词不能为空" |
| 调度器未运行 | 状态显示"未运行"，任务列表显示"暂无定时抓取任务" |
| 保留天数 < 1 | 阻止保存，提示"保留天数至少为 1" |

## 7. 关联功能

- **标签提取**（`/articles`）：设置页的提示词配置影响文章列表页的标签提取
- **消息任务**（`/message-tasks`）：消息任务在设置页的调度器区域显示状态
- **RSS 与消息任务文档**：`docs/features/rss-and-message-tasks.md`
