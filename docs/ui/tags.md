# 标签管理页面

## 面向谁

需要为文章打标签、对公众号内容进行人工分类的操作员。

## 解决什么问题

手动创建标签、管理已有标签的启用状态、删除不需要的标签，或通过 CSV 批量导入导出标签。

## 标签列表页

**访问路径：** `/tags`

### 页面元素

| 区域 | 内容 |
|------|------|
| 页面标题 | 标签管理 |
| 搜索框 | 按标签名称搜索，支持实时过滤 |
| 标签列表 | 展示标签名称、状态（启用/禁用/封禁）、文章数量、创建时间 |
| 操作列 | RSS 链接、编辑、删除按钮 |

### 如何理解"文章数量"

`文章数量`是该标签关联的文章总数（通过 `article_tags` 表统计）。显示为徽章样式，如 `12`。

点击标签名称旁边的 **RSS 图标**（`</>Rss`），可以在新标签页打开该标签的 RSS Feed，格式为 `/feed/tag/{tag_id}.atom`。

### 批量操作

- 勾选左侧复选框，可一次性删除多个标签
- 顶部会显示已选择数量，点击"批量删除"确认

### 状态切换

点击状态徽章（`启用` / `禁用` / `封禁`）可以直接切换状态：
- **绿色 "启用"**：标签正常使用
- **灰色 "禁用"**：标签暂时不可用
- **红色 "封禁"**：标签被锁定，不可切换

### 导出 / 导入

页面右上角提供"导出标签"和"导入标签"按钮：

- **导出**：生成 `标签列表.csv` 文件，包含所有标签的 id、名称、封面图、描述、状态、创建时间、关联公众号 ID
- **导入**：上传 CSV 文件，支持增量更新（id 相同则更新，不存在则新建）

## 标签编辑页

**访问路径：** `/tags/add`（新建）、`/tags/edit/{id}`（编辑）

### 可编辑的字段

| 字段 | 说明 |
|------|------|
| 标签名称 | 必填，2-30 个字符 |
| 封面图 | 上传图片作为标签封面，建议尺寸 120x120，支持 JPG/PNG，最大 2MB |
| 描述 | 标签的简介说明 |
| 状态 | 启用 / 禁用 / 封禁 |
| 关联公众号 | 可选择该标签关联哪些公众号（留空表示关联所有） |

### 表单提交

- 新建标签后需要手动刷新（点击"应用"按钮）任务才会生效
- 编辑后同样需要刷新

## 筛选功能

列表页搜索框支持实时过滤，输入关键词后列表会自动刷新到第 1 页。点击"清除"可恢复显示全部标签。

## 分页

页面底部显示总条数和当前页码，支持跳转到指定页。默认每页 10 条。

## 常见操作流程

**新建标签：**
1. 点击"添加标签"
2. 填写标签名称、封面、描述
3. 选择关联公众号（可选）
4. 点击"提交"
5. 返回列表页，点击"应用"按钮使改动生效

**批量导入标签：**
1. 在列表页点击"导入标签"
2. 选择 CSV 文件（需包含 `标签名称`、`状态`、`mps_id` 列）
3. 系统显示导入结果（导入数、更新数、跳过数）

**导出后编辑再导入：**
1. 点击"导出标签"获取 `标签列表.csv`
2. 在 Excel 中编辑（如修改描述、状态）
3. 点击"导入标签"上传修改后的文件
