# 订阅管理页

## 1. 页面作用

订阅管理页用于管理已添加的微信公众号订阅。用户可以查看订阅列表、查看订阅详情、刷新订阅（抓取最新文章）、编辑订阅信息、删除订阅，以及添加新订阅。

## 2. 入口路径

- 路径：`/subscriptions`
- 导航栏点击"订阅管理"进入

## 3. 页面结构

页面采用**左右布局**：

### 3.1 左侧：订阅列表

- **搜索框**：按公众号名称搜索
- **添加按钮**：跳转 `/add-subscription` 添加新订阅
- **订阅表格**：显示公众号名称、文章数量、状态
  - 点击行可选中该订阅，右侧显示详情
  - 选中行有高亮（左侧 4px 边框 + 浅色背景）
- **分页**：底部分页导航，支持跳转页

### 3.2 右侧：订阅详情

选中订阅后显示：

- **头部**：公众号头像、名称、ID、操作按钮
- **信息卡片**：
  - 状态（已启用/已禁用）
  - 文章数量
  - 最后同步时间
  - 时间跨度（最早文章 ~ 最新文章）
  - 简介
  - RSS 地址（有则显示）

- **操作按钮**：
  - **刷新**：立即抓取该订阅的最新文章（`/feed/mp/{mp_id}`）
  - **编辑**：弹出编辑弹窗
  - **删除**：确认后删除

### 3.3 空态

未选中订阅时，右侧显示"请从左侧列表选择一个订阅查看详情"。

## 4. 常见操作

### 4.1 刷新订阅

点击右侧"刷新"按钮，向后端发起抓取请求。后端调用微信采集接口，抓取该公众号的最新文章。

**限制**：同一订阅 60 秒内只能刷新一次。刷新过于频繁会返回 40402 错误，提示剩余等待时间。

### 4.2 查看订阅详情

点击订阅行，右侧显示该订阅的完整信息：
- 头像、名称、ID
- 状态、文章数、同步时间
- 简介
- RSS 链接（可点击跳转）

### 4.3 编辑订阅

点击"编辑"按钮，弹出 Dialog：
- 修改公众号名称
- 修改简介
- 切换启用/禁用状态
- ID 不可修改（只读）

### 4.4 删除订阅

点击"删除"按钮，弹出确认框。确认后删除订阅及其所有文章。

### 4.5 添加新订阅

点击左上角"添加订阅"按钮，跳转 `/add-subscription` 页面。

## 5. 注意事项

- 刷新操作有 60 秒防抖限制，防止频繁请求
- 删除订阅会级联删除该订阅下的所有文章
- RSS 地址由后端动态生成，格式为 `/feed/mp/{mp_id}.rss`
- 订阅状态为 `status=1` 表示启用，`status=0` 表示禁用
- 左侧列表选中状态通过 URL 参数同步（如 `/subscriptions/{mp_id}`）

## 6. 常见报错 / 空态

| 场景 | 表现 |
|---|---|
| 无订阅 | 左侧列表显示"暂无数据" |
| 刷新过于频繁 | Toast 提示"请等待 X 秒后再试" |
| 刷新失败 | Toast 显示错误信息 |
| 删除失败 | Toast 显示错误信息 |

## 7. 关联功能

- **添加订阅**（`/add-subscription`）：通过文章链接提取公众号、手动输入、导入/导出 CSV
- **文章列表**（`/articles`）：按公众号筛选文章
- **RSS**：`/feed/mp/{mp_id}.{format}` 获取该订阅的 RSS Feed
- **消息任务**：可以为订阅配置定时抓取任务，通过 `mps_id` 指定订阅
